# Проект 14: Место

### Обзор
* Описание проекта
* Технологии

**Описание проекта**

Это проект "Место". Интерактивная страница, куда можно добавлять фотографии, удалять их и ставить лайки.

**Технологии**

Для реализации данного проекта велась работа c новым материалом из тренажера Yandex.Praktikum.
